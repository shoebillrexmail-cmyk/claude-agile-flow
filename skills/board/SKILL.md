Read the Obsidian vault sprint board and backlog for the current project.

1. Find the project's vault path from the repo's CLAUDE.md under `## Obsidian Project`
2. Read `Sprint/Board.md` — list items per column (Ready, In Progress, In Review, Done)
3. Read `Backlog/Product-Backlog.md` — list items per column (Icebox, Needs Refinement, Refined)
4. Summarize: how many items in each column, what's actively in progress, what's ready to pick up
5. If there are story files referenced, read their **Points** and **Priority** to give a weighted view

Format the output as a clean status report.
